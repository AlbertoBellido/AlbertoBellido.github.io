Overview

The Student Registration System will be used by a local college to register students for courses and also for students to self-register. The program was built using Java libraries in a TextPad editor. The Bug Track System will be used to track application defects (i.e. bugs). The program was built using OpenShift an open source hybrid cloud application Platform as a Service (PaaS) developed by Red Hat, along with phpMyAdmin to handle the administration of MySQL database management system.

Screens

Various screens were designed for the user to initiate and complete the registration process as Login Screen, Create Account Screen, Course Registration Screen, Profile Screen, and Navigation Option Screen. In the same manner for the Bug Track System such as Defect Tracker Screen, View Open Ticket Screen, View Ticket Screen, and Update Ticket Screen.
