Overview

The Student Registration System will be used by a local college to register students for courses and also for students to self-register. The program was built using Java libraries in a TextPad editor.

Screens

A few screens were designed for the user to be able to initiate and complete the registration process as Login Screen, Create Account Screen, Course Registration Screen, Profile Screen, and Navigation Option Screen.
